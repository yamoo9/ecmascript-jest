테스트 주도 개발 (DOM Test)

Happy DOM은 그래픽 사용자 인터페이스가 없는 웹 브라우저의 JavaScript 구현입니다. 여기에는 WHATWG DOM 및 HTML의 많은 웹 표준이 포함됩니다.

Happy DOM의 목표는 테스트, 웹 사이트 스크래핑(scraping) 및 서버 사이드 렌더링(server-side rendering)에 유용하도록 웹 브라우저를 에뮬레이트하는 것입니다. Happy DOM은 성능에 중점을 두고 있으며 JSDOM의 대안으로 사용할 수 있습니다. 이 패키지를 사용하면 Jest와 함께 Happy DOM을 사용할 수 있습니다.
